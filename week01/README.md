# Week 01 - C++ Cafe

## 목표

이번 실습에서는 C++ 프로그램의 기본 구조를 익힙니다.

사용하는 개념:

- `#include <iostream>`
- `int main()`
- 변수
- `std::cin`
- `std::cout`
- `if` / `else`

## 실행 방법

Codespaces 터미널에서 다음 명령을 실행하세요.

```bash
cd week01
make run
```

## 자기 점검

다음 명령을 실행하세요.

```bash
make check
```

성공하면 다음과 같은 메시지가 나옵니다.

```text
All checks passed.
Checkpoint code: W01-CPP-CAFE-PASS
```

## 확인 질문

2. `price` 변수는 무엇을 저장하나요?
3. `quantity <= 0` 조건은 왜 필요한가요?
4. `std::cin`과 `std::cout`의 역할은 무엇인가요?
